# @vehiverze/shared-utils

Shared utility functions for the Vehiverze monorepo.

## Available Utilities

- `cn` - Class name utility for merging Tailwind classes
- `formatDate` - Date formatting utility

## Usage

```typescript
import { cn } from "@vehiverze/shared-utils/cn";
import { formatDate } from "@vehiverze/shared-utils/format-date";
```
