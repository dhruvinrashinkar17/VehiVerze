# @vehiverze/ui

Shared UI components for the Vehiverze monorepo based on shadcn/ui.

## Usage

Import components from this package in your apps:

```typescript
import { Button } from "@vehiverze/ui/button";
import { Card } from "@vehiverze/ui/card";
```

## Components

This package includes all shadcn/ui components used across the monorepo.
